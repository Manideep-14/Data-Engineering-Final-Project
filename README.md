# Data-Engineering-Final-Project
Supply Chain Performance Analysis Using Databricks

Project Overview

This project analyzes supply chain data using Databricks and the Medallion Architecture (Bronze, Silver, Gold). The objective is to generate business insights related to profitability and delivery performance through data processing and dashboard visualization.

Business Objectives

1. Profit Analysis

Analyze the average profit generated by different product categories to identify the most profitable categories and support business decision-making.

2. Delivery Performance Analysis

Analyze delivery status trends, including late, on-time, advance shipping, and cancelled orders, to evaluate supply chain efficiency and improve delivery performance.

Architecture

The project follows the Medallion Architecture:

* Bronze Layer – Raw data ingestion
* Silver Layer – Cleaned and transformed data
* Gold Layer – Business-ready analytical views

Technologies Used
